phpstorm食用指南(自用)

2022-12-05

phpstorm

快捷键

  • Ctrl + Alt + Left ———— 一步一步的退回到之前的地方。（ctrl跳转）

  • Ctrl + Shift + C ———— 复制光标选中代码的绝对文件路径

  • Shift + Alt + C ———— 复制光标所在代码的相对路径(包含行号) 更实用!!

  • Ctrl + Shift + N ———— 打开快速跳转窗口 可以填入以上俩个命令复制出来的路径或者类名 快速打开所在文件 （用于同事交流或者自己做流程图架构图的时候使用 直接把代码具体到行号给他 他自己打开）

  • 双击Shift ———— 功能同上条，打开tab不同

  • Ctrl + G ———— 跳转到指定行

  • Ctrl + B ———— 跳转引用

  • Ctrl + [左右方向] ———— 左右跳转到下一个语句点

  • Ctrl + D ———— 复制+粘贴

  • Ctrl + W ———— 扩充选中点

  • Shift + End ———— 当前光标选中到行末

  • Ctrl + Shift + [左右方向] ———— 当前光标往左右选中下一个语法点

  • Ctrl + Shift + [+] ———— 当前打开文件，代码全部展开

  • Ctrl + Shift + [-] ————当前打开文件，代码全部折叠

  • Ctrl + [+] ———— 当前光标所在地，代码展开

  • Ctrl + [-] ———— 当前光标所在地，代码折叠

  • Ctrl + Shift + U ———— 大小写切换

  • 光标选中数据 + Shift + [" ( [ 等符号] ———— 自动把字符串包括起来

  • Ctrl + Shift + R ———— 全局替换

  • Ctrl + Shift + F ———— 全局搜索

  • F5 ———— 复制

  • Shift + F6 ———— 重构命名
