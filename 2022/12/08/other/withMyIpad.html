我与我的ipad(ipad当win10副屏之路)

2022-12-08

Ipad副屏之路

背景

关于我

  • 身份：穷
  • 物品：台式i5-4代老破旧 + ipad 2019 小卑微
  • 习惯：公司双屏working
  • 梦想：在家里也能过上双屏的日子

尝试

ScreenTask

  • 软件来源：github
  • 技术思路：脚本不断截屏，监听http server，ipad打开网页 不断刷新读取server
  • 效果体验：一颗星，用来看小说还行，看视频就die了，更别说写代码

虚拟屏幕驱动 + OBS推流

虚拟屏幕驱动 https://www.amyuni.com/forum/viewtopic.php?t=3030

  • 软件来源：OBS官网 + 虚拟屏幕驱动（付费买了一个）
  • 技术思路：实现一个内网直播，直播指定的虚拟屏幕，然后内网观看
  • 效果体验：一颗星，视频解码过慢，不是网络传输问题，最小延迟1~2秒。用来糊弄女朋友还行，让她在床上看一些电脑播放的视频吧

虚拟屏幕驱动 + webRTC

  • 软件来源：一开始尝试自己写webRTC,后面在github找到一款screego
  • 技术思路：webRTC P2P分享屏幕
  • 效果体验：三颗星，不错不错不错！至少可用了

串流

  • 软件来源：百度 github
  • 技术思路：利用GeForce 或者stream等平台 + 平板上月光软件(moonlight)
  • 效果体验：俩颗星，还不错，但是我这边可能使用的不是显卡欺骗器而是软件虚拟屏幕驱动，没办法像B站教程一样 实现副屏效果，只能同屏复制

spacedesk

  • 软件来源：百度 github
  • 技术思路：不知道，现成的免费软件，有ipad客户端
  • 效果体验：三颗星，还不错 可用

其他软件

未测试

  • Splashtop Wired XDisplay - 可以试用
  • Duedisplay - 付费
