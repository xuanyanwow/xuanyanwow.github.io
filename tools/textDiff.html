文本对比

超级强的文本对比工具
使用场景:
* 报名名单比较
* url传参比较排查
* JSON结构差异比较




文本A 文本B
{{i + 1}} {{diffResultText(item,'left')}} {{i + 1}} {{diffResultText(item,'right')}}






使用示例

先按行排序

用途：报名名单、订单资料比较等

场景：部门活动让员工填写电子表格报名（员工填写顺序不固定），需要找出还未报名的同事。

操作：填入固定的部门名单，右侧填入员工报名的电子表格；先排序 再对比 即可比较出还未报名的员工名单

先按符号分割

场景：对接接口的时候，同事调不通，可以对比一下参数哪里有差异。

操作：填入query string，如 name=siam&age=18 按 & 分割，即可让参数一行一条 方便查看对比

json/url结构比较

场景：如上，对接接口 对比参数差异。但是只关系结构（key是否存在）而不在意内容（比如每次调用接口的sign都不一样）

示例数据1

{
    "name":"siam",
    "friend": {
        "name":"friend1",
        "age":18
    }
}
{
    "name":"小明",
    "friend": {
        "name":"friend2",
        "age":18
    }
}

示例数据2

name=siam&money=189789&sign=jklj3k2lj535980al
name=siam&money=9871526&sign=j329879zj3432421l
